# Betracard - Trust Scoring Platform

A modern, interactive marketing website for Betracard, Africa's leading Trust Infrastructure company providing dynamic trust scoring solutions.

## Tech Stack

- **Next.js 14+** with App Router
- **TypeScript** for type safety
- **TailwindCSS** for styling
- **Framer Motion** for advanced animations
- **Lucide React** for icons

## Features

- **Interactive Animations**: Smooth hover effects, scaling, rotations, and scroll-triggered animations
- **Dynamic Trust Scoring Focus**: Emphasizes trust scores over basic verification
- **Responsive Design**: Mobile-first approach with flexible layouts
- **Modern UI/UX**: Gradient backgrounds, glassmorphism effects, and engaging micro-interactions
- **Real-time Elements**: Animated trust score displays and interactive components

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
